# Code Golf Minifier

Minify C/C++ and JavaScript code for code golf while preserving behavior.

## Features

- Removes comments and unnecessary whitespace.
- Preserves string/char/raw string literals and JavaScript regex/template literals.
- Keeps C/C++ preprocessor directives valid.

## Command

- `Code Golf: Minify C/C++` (`Ctrl+Alt+M` / `Cmd+Alt+M`)
- `Code Golf: Minify JavaScript`

## Notes

This is a lightweight tokenizer-based minifier. If you hit edge cases, open an issue or PR.
